Box Model CSS

El Box Model (modelo de caja) es el fundamento de cómo CSS calcula el espacio que ocupa cada elemento en una página web.

Las 4 capas del Box Model

Margin
Border
Padding
Content
(Width × Height)

Componentes del Box Model (de dentro a fuera):

Capa Descripción Ejemplo
Content El área donde se muestra el contenido del elemento. width: 200px, height: 100px
Padding El espacio entre el contenido y el borde del elemento. padding: 10px
Border El borde que rodea al elemento. border: 2px solid #000
Margin El espacio entre el borde y los elementos vecinos. margin: 10px

Buenas Prácticas

Algunas recomendaciones para trabajar con el Box Model:

  • Utiliza box-sizing: border-box; para incluir padding y border en el ancho y alto del elemento.
  • Evita usar margenes grandes que puedan afectar el diseño responsive.
  • Considera el uso de unidades relativas (como em o rem) para un mejor control del tamaño del texto.
  • Usar padding para espacio interno y margin para espacio externo. Recordar que margin no tiene color de fondo.
