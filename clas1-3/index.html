Clase 3: Selectores de clase.

Una clase es un identificador que se puede asignar a uno o más elementos HTML.

Los selectores de clase se definen con un punto (.) seguido del nombre de la clase. Se pueden aplicar a cualquier elemento HTML.

Sintaxis:

Estilos css:

            .nombre-de-clase {
                propiedad: valor;
            }

Html:

            <elemento class="nombre-de-clase">Contenido</elemento>

Tipos de selectores de clase

Existen diferentes tipos de selectores de clase, como:

Selector universal 
* {color: black;}
 Aplica el estilo a todos los elementos del documento.
Selector de tipo 
p {color: red;}
 Aplica el estilo a todos los elementos del tipo especificado (en este caso, a todos los párrafos).
Selector de clase 
.fondo {background-color: lightgray;}
 Aplica el estilo a todos los elementos que tengan la clase especificada (en este caso, a todos los elementos con la clase "fondo").
Selector de ID 
#mi-id {font-size: 20px;}
 Aplica el estilo al elemento que tenga el ID especificado (en este caso, al elemento con el ID "mi-id").
Selector descendiente 
div p {color: green;}
 Aplica el estilo a todos los elementos del tipo especificado que están dentro de otro elemento del tipo especificado (en este caso, a todos los párrafos que están dentro de un div).
Selector de HUD DIRECT 
div > p {color: blue;}
 Aplica el estilo a todos los elementos del tipo especificado que son hijos directos de otro elemento del tipo especificado (en este caso, a todos los párrafos que son hijos directos de un div).

Pseudo-clases

Las pseudo-clases son palabras clave añadidas a los selectores que especifican un estado special del elemento seleccionado. Se escriben después del selector, precedidas por dos puntos (:).

Ejemplo:

.enlace:hover {color: orange; background-color: lightblue; text-decoration: underline; cursor: pointer;}

En este ejemplo, el estilo se aplicará a los elementos con la clase "enlace" cuando el usuario pase el cursor sobre ellos.

Pseudo-clases comunes:

.enlace:link {color: blue;}
 Aplica el estilo a los enlaces que aún no han sido visitados. 
.enlace:visited {color: purple;}
 Aplica el estilo a los enlaces que ya han sido visitados. 
.enlace:hover {color: orange;}
 Aplica el estilo a los elementos cuando el usuario pasa el cursor sobre ellos. 
.enlace:active {color: red;}
 Aplica el estilo a los elementos en el momento en que son activados (por ejemplo, al hacer clic en un enlace). 
.enlace:focus {color: green;}
 Aplica el estilo a los elementos que tienen el foco (por ejemplo, al hacer clic en un campo de formulario).

Especificidad

La especificidad es un mecanismo que determina qué reglas de estilo se aplican a un elemento cuando hay varias reglas que podrían aplicarse. La regla con mayor especificidad tiene prioridad sobre las demás.

La especificidad se calcula asignando un valor numérico a cada tipo de selector:

  • Importante (!important): 1000
  • Selector de estilo en línea: 1000
  • Selector de ID: 100
  • Selector de clase, pseudo-clase o atributo: 10
  • Selector de tipo o pseudo-elemento: 1
  • Selector universal: 0
