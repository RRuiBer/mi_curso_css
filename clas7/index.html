Metodología BEM

BEM (Block, Element, Modifier) es una metodología para nombrar clases en CSS de manera que sea más fácil de entender y mantener. BEM ayuda a crear componentes reutilizables y a evitar conflictos de nombres en el CSS.

Estructura de BEM

Sigla Componente Descripción Ejemplo
B Bloque Un bloque es un componente independiente y reusable. .header, .menu, .card
E Elemento Un elemento es una parte de un bloque que no tiene un significado independiente. .header__title, .menu__item, .card__image
M Modificador Variación o estado de un bloqueo un elemento. .menu--active, .header--large, .menu__item--disabled

Sintaxis de BEM

📄 styles.css

/* BLOCK */
.bloque {}

/* ELEMENT (con doble guión bajo) */
.bloque__elemento {}

/* MODIFIER (con doble guión) */
.bloque--modificador {}
.bloque__elemento--modificador {}
