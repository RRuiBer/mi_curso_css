Posicionamiento CSS

El posicionamiento en CSS permite controlar la ubicación de los elementos en una página web. Existen diferentes tipos de posicionamiento que se pueden aplicar a los elementos HTML.

Tipos de Posicionamiento

Tipo Descripción Uso común
Static Es el valor por defecto. Los elementos se posicionan de acuerdo al flujo normal del documento. Elementos normales
Relative El elemento se posiciona en relación a su posición original en el flujo del documento. Punto de referencia para otros elementos
Absolute El elemento se posiciona en relación a su contenedor más cercano que tenga un posicionamiento distinto a static. Menús, tooltips, overlays
Fixed El elemento se posiciona en relación a la ventana del navegador y permanece fijo al hacer scroll. Headers, footers, botones de navegación
Sticky El elemento se comporta como relative hasta que alcanza un punto específico, luego se comporta como fixed. Headers que se quedan fijos al hacer scroll, tablas de contenido
